How to create AWS Account 
=========================
We required following things to create AWS account.

1) Gmail Id 

2) Debit/Credit Card 

3) Minimum 2 rupees must available in account

Note:
-----
If we use debit card to create an account then it takes less then 24 hours to activate the account.

If we use credit card to create an account then it takes less then 2 to 3 hours to activate the account.

History of Linux 
================
Linux operating system developed by Linus Torvalds.

The idea behind linux is why to go for licence to use any operating system.

He has started one open source project called linux.

He created linux kernel.

Kernel is a first layer of operating system.

Linux kernel is used to interact with hardware components.

The main aspect to create linux are

1) It create free CLI OS 

2) It should be open source

3) It should run on any hardware

4) GNUX (Generally Not a Unix)

Today we have 300+ distributions in linux.


EC2
====
EC2 stands for Elastic Cloud Compute.

It is a webservice provided by AWS to create virtual servers(instances) in a cloud. 

EC2 comes in Infrastructure As a Service (IaaS).

Linux provides two ways authentication.

1) Username and password (Recommanded)

2) Username with key (Not Recommanded)
