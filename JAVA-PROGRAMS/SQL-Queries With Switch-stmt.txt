Q) Write a java program to find out given alphabet is a vowel or consonent?

Approach1:
----------
import java.util.Scanner;
class Test  
{
	public static void main(String[] args) 
	{
		Scanner sc = new Scanner(System.in);
		System.out.println("Enter the alphabet :");
		char ch = sc.next().charAt(0);
		
		switch(ch)
		{
			case 'a','e','i','o','u' -> 
				{System.out.println("It is a vowel");}
			default -> 
				{System.out.println("It is a consonent");}
		}
	}
}

Approach2
---------
import java.util.Scanner;
class Test  
{
	public static void main(String[] args) 
	{
		Scanner sc = new Scanner(System.in);
		System.out.println("Enter the alphabet :");
		char ch = sc.next().charAt(0);
		
		String result = switch(ch)
		{
			case 'a','e','i','o','u' -> { yield "It is a vowel"; }
			default -> { yield "It is a consonent"; }
		};
		
		System.out.println("Your Result : "+result);
	}
}

Q) Write a query to display distinct records from product table?

Q) Write a query to display duplicates records from product table?

select * from product group by pid,pname,pprice having count(*)>1 order by pid;


Q) Write a query to remove duplicate records from product table?

	delete from product where rowid  NOT IN 
	(select min(rowid) from product group by pid,pname,pprice);


Q) Write a query to display highest price product from product table?

	select * from product where pprice = (select max(pprice) from product);
