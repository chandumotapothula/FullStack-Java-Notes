CSS Pseudo Elements ::before ::after 
================================
A CSS pseudo-element is used to style specific parts of an element. 


ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!</title>
	<style>
		h1
		{
			text-align: center;
			margin: 100px auto;
		}
		h1::before
		{
			content: "Ihub";
			color:red;
		}
		h1::after
		{
			content: "Institute";
			color:blue;
		}
	</style>
</head>
<body>
		<h1> Talent </h1>
</body>
</html>

History of JavaScript 
=====================
Original name JavaScript is LiveScript.

LiveScript was developed by Netscape Corporation in the year of 1991.

In 1995,Branden Eich a popular scientist of Netscape Corporation  renamed LiveScript to JavaScript.

The official name of JavaScript is ECMA Script.

ECMA stands for Europeon Company Manufacturer Association.




Advantages of JavaScript 
========================
1) It is used to develop interactive web pages.

2) It is used to display dialog boxes and pop-up boxes.

3) It is used to perform client side form validation.

4) It is used to add dynamic content in our web page.

5) It supports objects like Arrays, Strings, RegularExpression, Date and etc.

6) It supports cookies.

7) It is easy to learn and use.

8) Rich User Interfaces.

9) Asynchronous operations.

and etc.



JavaScript Syntax
================
<script type="text/javascript"   lang="javascript">
	-
	- // set of statements 
	-
</script>

Note:
-----
	Here type and lang attributes are optional.

ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!</title>
</head>
<body>
		<script type="text/javascript" lang="javascript">
			
			document.write("Welcome to JavaScript");

		</script>
</body>
</html>

Note:
-----
	Markup language we can't use in scripting language directly. 
	To overcome that limitation we need to use below code.
ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!</title>
</head>
<body>
		<script>
			
			document.write("Welcome to JavaScript")
			document.write("<br>");
			document.write("This is Niyaz sir class")

		</script>
</body>
</html>

Note:
-----
	We can declare HTML and CSS code inside Javascript as given below.
ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!</title>
</head>
<body>
		<script>
			
		document.write("<h1 style='text-align:center'>Welcome to JavaScript</h1>")
			document.write("<br>");
			document.write("<p style='font-size:50px;'>This is Niyaz sir class</p>")

		</script>
</body>
</html>

Note:
----
	If we keep HTML, CSS and JavaScript is a single place then it is called DHTML.
	DHTML stands for Dynamic Hypertext Markup Language.

Comments in JavaScript 
======================
Comments are created for documentation purpose.

Comments are used to improve readability of our code.

JavaScript engine does not display comments in a web page or a browser.

We have two types of comments in JavaScript.

1) Single Line Comment 
---------------------
	It is used to comment a single line.
	ex:
		// comment here 

2) Multiple Line Comment 
--------------------
	It is more convenient because we can comment a single line as well as multiple lines.
	ex:
		/*
			-
			-  comment here 
			-
		*/
