CSS clear property 
===================

The clear property specifies on which side of an element floating elements are not allowed to float.

value		Description
-------		----------------
none		Default allows floating elements to both the side.

left		No floating elements allowed on the left side.

right 		No floating elements allowed on the right side.

both		No floating elements allowed on the both side.

ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!</title>
	<style>
		.box1
		{
			width:200px;
			height:200px;
			background-color:#d63031;
			float:left;	
		}
		.box2
		{	
			width:200px;
			height:200px;
			background-color:#6c5ce7;	
			float:right;
		}
		.box3
		{
			width:100%;
			height:200px;
			background-color:#ffeaa7;
			clear:none;
		}
	</style>
</head>
<body>
	<div class="box1"></div>
	<div class="box2"></div>
	<div class="box3"></div>	
</body>
</html>

CSS List

The CSS list properties allow you to:

> Set different list item markers for ordered lists
> Set different list item markers for unordered lists
> Set an image as the list item marker
> Add background colors to lists and list items

CSS display property 
====================
The display property spcifies the display behaviour of an element.

syntax
--------
	display: value;

value		description
---------	-------------------
none		The element is completely removed.

inline 		displays an element as on inline element.
		Any height and width properties will not effect.

block		Displays an element as block element.
		IT starts on a new line and takes up the whole width.

inline-block	displays an element as an inline-level container.
		The element itself is formatting  as an inline element.
		but we can apply height and width values.

inherit		Inherits this property from its parent element.


ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!</title>
	<style>
		h1
		{
			display: none;
		}
		p
		{
			display: none;
		}
	</style>
</head>
<body>
		<h1> Heading Tag </h1>
		<p> Paragraph Tag </p>
</body>
</html>

box-sizing  property
=====================
The box-sizing property defines how the width and height of an element are calculated: should they include padding and borders, or not.


box-sizing:content-box
----------------------------
It will take seperate width,border and padding.But it will not take  margin.

ex:
----
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!</title>
	<style>
		div
		{
			height: 200px;
			width:200px;
			background-color: #FFFF00;
			border:2px solid black;
			padding: 10px;
			box-sizing: content-box;
		}
	</style>
</head>
<body>
		<div>  </div>
</body>
</html>


box-sizing:border-box
---------------------------
It includes width,border and padding.But it will not include marging.

ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!</title>
	<style>
		div
		{

			border:2px solid black;
			padding: 10px;
			box-sizing: border-box;
		}
	</style>
</head>

		<div>  </div>
</body>
</html>
