JavaScript Spread Operator 
=========================
The JavaScript spread operator ( ... ) allows to spread the array, to copy array/object to modify array/object and to merge array/object.

ex:1
-----
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>

			function f1(arg1,arg2,arg3)
			{
				document.writeln(arg1+arg2+arg3);
			}
			var arr = [10,20,30];
			f1(...arr);
		</script>
	</body>	
</html>

Q) Explain Synchronous and Asynchronous in JavaScript?


Synchronous
------------
Synchronous JavaScript is a programming approach that enables the blocking execution of tasks, allowing sequence operations, degrading responsiveness, and non-efficient handling of time-consuming operations in web applications.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
		<script>
			document.writeln("First Stmt");
			document.writeln("<br>");
			document.writeln("Second Stmt");
			document.writeln("<br>");
			document.writeln("Third Stmt");
		</script>
	</body>	
</html>

Asynchronous 
------------
Asynchronous JavaScript is a programming approach that enables the non-blocking execution of tasks, allowing concurrent operations, improved responsiveness, and efficient handling of time-consuming operations in web applications.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
		<script>
			document.writeln("First Stmt");
			document.writeln("<br>");
			setTimeout(function(){
				document.writeln("Second Stmt");
			},3000);
			document.writeln("<br>");
			document.writeln("Third Stmt");
		</script>
	</body>	
</html>
