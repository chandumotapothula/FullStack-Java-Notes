setInterval()
==============
A setInterval() execute the function repeatedly.

The setInterval() method calls a function at specified intervals (in milliseconds).

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			img
			{
				width:200px;
				height:200px;
			}
		</style>
	</head>
	<body>
			<img src="images/car.gif" id="cid"/>
			<br>
			<button onclick="f1()"> Start Race </button>

			<script>
				function f1()
				{
					var a = 0;
					var target = document.getElementById('cid');
					setInterval(f2,1000);
					function f2()
					{
						a = a + 10;
						target.style.marginLeft=a+"px";
					}

				}
			</script>
	</body>	
</html>


clearInterval()
================
The clearInterval() method clears a timer set with the setInterval() method. 

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			img
			{
				width:200px;

	</head>
	<body>

				{
					var a = 0;

					var myId = setInterval(f2,1000);
					function f2()

	</body>	
</html>


localStorage 
============
A localStorage properties allows us to save key/value pairs in a browser window.

A localStorage allows us to store the data with no-expiry.It means our data will not be 
delete even if we close the browser.It will be present for next day.

A localStorage is a read-only.

To add the data in a localStorage we need to use setItem(key,value) function.

To read the data from localStorage we need to use getItem(key) function.

To remove perticular data from localStorage we need to use removeItem(key) function.

To remove all the data from localStorage we need to use clear() function.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
			<script>
				localStorage.setItem("name","Alan");
				localStorage.setItem("age",32);
				document.writeln(localStorage.getItem("name")+"<br>");
				document.writeln(localStorage.getItem("age")+"<br>");
				localStorage.removeItem("age");
				document.writeln(localStorage.getItem("name")+"<br>");
				document.writeln(localStorage.getItem("age")+"<br>");
				localStorage.clear();
				document.writeln(localStorage.getItem("name")+"<br>");
				document.writeln(localStorage.getItem("age")+"<br>");

			</script>
	</body>	
</html>

===============

A sessionStorage store the data with respect to one session.It means our data will be deleted
once if we close the browser window.

To add the data in a sessionStorage we need to use setItem(key,value) function.

To read the data from sessionStorage we need to use getItem(key) function.

DOM 
====
DOM stands for Document Object Model.

Whenever HTML document is loaded in a browser it represents DOM.

DOM is a tree node structure.

A document is a root node for entire HTML document.

DOM always looks for three nodes.

1) Element Node 

2) Attribute Node 

3) Text Node 

Diagram: frontend21.1

Document object is used to add dynamic content in a web page.

A document object contains following functions.

1) document.write()

2) document.writeln()

3) document.getElementById()

4) document.getElementsByName() 

5) document.getElementsByTagName()

6) document.getElementsByClassName()

7) document.addEventListener()

8) document.removeEventListener() 

1) document.write()
--------------------
It is used to declare custom messages or data without space at the end.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
			<script>
				document.write("IHUB");
				document.write("TALENT");
			</script>
	</body>	
</html>

3) document.getElementById()
--------------------------
It is used to read the elements by using specified id.
If element is not found then it will return null.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
			Name: <input type="text" id="t1"/> 
			<button onclick="f1()"> submit </button>
			<script>
				function f1()
				{
					var name = document.getElementById('t1').value;
					document.writeln("Welcome :"+name);
				}
			</script>
	</body>	
</html>

ex:
---
<!DOCTYPE html>
