4) document.getElementsByName() 
===============================
It is used to select the elements based on specified name.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
			Name:  <input type="text" name="t1"/> 
			<button onclick="f1()"> submit </button>

			<script>
				function f1()
				{
					var name = document.getElementsByName('t1')[0].value;
					document.writeln("Welcome :"+name);
				}
			</script>
	</body>	
</html>

How to select multiple checkboxes using javscript
-------------------------------------------------
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
			Course Completed?
			<br>	
			<button onclick="f1()"> select all </button>
			<br><br>
			<input type="checkbox" name="t1" value="html"/> HTML <br>
			<input type="checkbox" name="t1" value="css"/> CSS <br>
			<input type="checkbox" name="t1" value="js"/> JavaScript <br> 
			<script>
				function f1()
				{
					var x = document.getElementsByName('t1');
					for(var i=0;i<x.length;i++)
					{
						if(x[i].type=="checkbox")
						{
							x[i].checked=true;
						}
					}
				}
			</script> 
	</body>	
</html>

6) document.getElementsByClassName() 
====================================
It is used to select the elements based on specified class name.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			.mystyle
			{
				width:200px;
				height:200px;
				background-color:red;
			}
		</style>
	</head>
	<body>
			<div class="mystyle"></div>
			<button onclick="f1()"> click me </button>
			<script>
				function f1()
				{
			var len = document.getElementsByClassName('mystyle').length;
			document.writeln(len);
				}
			</script>
	</body>	
</html>

ex:
---
<!DOCTYPE html>

7) document.addEventListener()
==============================
It is used to add the event listener to HTML element in a DOM.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
			<h2> Click Anywhere </h2>
			<script>
				document.addEventListener('click',function(){
					alert("You have clicked");
				})
			</script>
	</body>	
</html>

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
			<h2 id="hover"> Mouse Over Here  </h2>
			<script>
				var x = document.getElementById('hover');
				x.addEventListener('mouseover',function(){
					alert("Mouse Is Over");
					})
			</script>
	</body>	
</html>
