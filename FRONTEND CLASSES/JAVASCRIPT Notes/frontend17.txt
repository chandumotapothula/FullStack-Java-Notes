3) If Else If stmt 
------------------
It is used to evaluate the code based on multiple conditions.

syntax:
------
	if(cond1)
	{
		- //code to be execute if cond1 is true 
	}
	else if(cond2)
	{
		- //code to be execute if cond2 is true 
	}
	else if(cond3)
	{
		- //code to be execute if cond3 is true 
	}
	else
	{
		- //code to be execute if all conditions are false.
	}

ex:
----
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
		<script>
			var n = parseInt(prompt("Enter the option :"));
			if(n==100)
				document.write("It is a police number");
			else if(n==103)
				document.write("It is a enquiry number");
			else if(n==108)
				document.write("It is emergency number");
			else
				document.write("Invalid option");

		</script>
	</body>	
</html>

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>

			else if(ch=='i')

				document.write("It is a vowel");

	</body>

JavaScript switch case 
======================
It is used to evaluate the code based on multiple conditions.
It is similar to if else if stmt.
A switch case used to read options from the enduser.
syntax:
------
	switch(condition/expression)
	{	
		case value1: //code to be execute
			     break stmt;	
		case value2: //code to be execute
			     break stmt;

		-
		-
		default:  //code to be execute if all cases are false.
	}
ex:
--
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
		<script>
			var n = parseInt(prompt("Enter the number"));
			switch(n)
			{
				case 100: 	document.write("It is police number");
					  		break;
				case 103: 	document.write("It is enquiry number");
					  		break;
				case 108: 	document.write("It is emergency number");
					  		break;
				default :   document.write("invalid option");
			}

		</script>
	</body>	
</html>

Note:
-----
Declaration of break stmt is optional in switch case . But if we won't declare break statement then from where our condition is satisfied from there all cases will be executed.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
		<script>
			var n = parseInt(prompt("Enter the option"));
			switch(n)
			{
				case 100: 	document.write("It is police number <br>");
					  		//break;
				case 103: 	document.write("It is enquiry number <br>");
					  		//break;
				case 108: 	document.write("It is emergency number <br>");
					  		//break;
				default :   document.write("invalid option");
			}

		</script>
	</body>	
</html>

JavaScript Loops 
=================
We have four types of loops in JavaScript.

1) do while loop

2) while loop 

3) for loop 

4) for in loop 

1) do while loop
------------------
It evaluates the code how long our condition is true.

syntax:
------
	do
	{	
		-
		-//code to be execute
		-
	}while(condition);

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
		<script>
			var i=1;
			do
			{
				document.write(i+" "); // infinite 1 
				
			}while(i<=10);
		</script>
	</body>	
</html>

ex:
