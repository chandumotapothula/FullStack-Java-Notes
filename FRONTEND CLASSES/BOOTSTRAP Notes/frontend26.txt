Abstraction 
===========
Hiding the internal implementation and highlighting the set of services is called abstraction.

In JavaScript, Abstraction often we can achieve using functions and classes that encapsulate behaviour by exposing neccesarry part to outside world.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
		<script>
			
			class Circle
			{
				//private field 
				#radius;

				constructor(radius)
				{
					this.#radius=radius;
				}

				getArea()
				{
					return Math.PI * this.#radius * this.#radius;
				}
			}
			var c = new Circle(5);
			document.writeln(c.getArea());
			
		</script>
	</body>	
</html>

Inheritance 
============
Inheritance is a mechanism in which one class will inherits the properties of another class.

The main objective of inheritance is to achieve reusability.

We have following list of inheritance in javascript.

1) Single Level Inheritance 

2) Multi Level Inheritance 

3) Multiple Inheritance 

4) Hierarchical Inheritance 

5) Hybrid Inheritance 

1) Single Level Inheritance
----------------------------
If we derived a class in the presence of one base class is called single level inheritance.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
		<script>
			
			class A
			{
				f1()
				{
					document.writeln("f1-function<br>")
				}		
			}
			class B extends A 
			{
				f2()
				{
					document.writeln("f2-function<br>")
				}		
			}
			var a = new A();
			a.f1();
			var b = new B();
			b.f1();
			b.f2();

			
		</script>
	</body>	
</html>

2) Multi Level Inheritance 
---------------------------
If we derived a class in the presence one base class and that class is derived from another base class is called multi level inheritance.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
		<script>
			
			class A
			{
				f1()
				{
					document.writeln("f1-function<br>")
				}		
			}
			class B extends A 
			{
				f2()
				{
					document.writeln("f2-function<br>")
				}		
			}
			class C extends B
			{
				f3()
				{
					document.writeln("f3-function<br>")
				}		
			}
			var a = new A();
			a.f1();
			var b = new B();
			b.f1();
			b.f2();
			var c = new C();
			c.f1();
			c.f2();
			c.f3();
			
		</script>
	</body>	
</html>

3) Multiple Inheritance
-----------------------
If we derived a class in the presence of multiple base classes is called multiple inheritance. 

JavaScript does not support multiple inheritance.


4) Hierarchical Inheritance
---------------------------
If we derived multiple classes using one base class is called hierarchical inheritance.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
		<script>
			
			class A 
			{
				f1()
				{
					document.writeln("f1-function <br>");
				}
			}
			class B extends A 
			{
				f2()
				{
					document.writeln("f2-function <br>");
				}
			}
			class C extends A 
			{
				f3()
				{
					document.writeln("f3-function <br>");
				}
			}
			var a  = new A();
			a.f1();
			var b = new B();
			b.f1();
			b.f2();
			var c = new C();
			c.f1();
			c.f3();


		</script>
	</body>	
</html>

5) Hybrid Inheritance
---------------------
It is a combination of more than two inheritance.

JavaScript does not support hybrid inheritance.




Polymorphism
============
Polymorphism has taken from Greek Word.

Here poly means many and morphism  means forms.

The ability to represent in different forms is called polymorphism.

The main objective of polymorphism is to provide flexibility.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
		<script>
			
			class A 
			{
				f1()
				{
					document.write("0-arg fun");
				}
				f1(arg1)
				{
					document.write("1-arg fun");
				}
				f1(arg1,arg2)
				{
					document.write("2-arg fun");
				}
			}
			var a = new A();
			a.f1();

		</script>
	</body>	
</html>

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
		<script>
			
			class A 
			{
				f1()
				{
					document.write("A-class <br>");
				}
			}
			class B extends A 
			{
				f1()
				{
					document.write("B-class <br>");
				}
			}
			var a = new A();
			a.f1(); // A-class
			var b = new B();
			b.f1(); // B-class
			

		</script>
	</body>	
</html>

JavaScript querySelector() 
==========================
The querySelector() method in JavaScript is a powerful tool for selecting elements within the Document Object Model (DOM) using CSS selectors. 

It is essential for interacting with the Document Object Model (DOM) and manipulating web page elements.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
			<h1> My Heading Tag </h1>

			<script>	
				var h = document.querySelector('h1');
				h.innerHTML = "My New Heading Tag";
			</script>
	</body>	
</html>

Bootstrap5
==========
It is a most popular HTML, CSS and JavaScript framework developed by Twitter to develop web applications.

It is a frontend framework developed by twitter to develop web applications rapidly and easily.

It was developed by Mark Otto and Jacob Thornton at Twitter.

It contains HTML and CSS based designed templates for typography, Forms, Tables, List and etc.

It was released to the public on August 19, 2011 on GitHUB.

It contains javascript plugins along with Jquery.

Bootstrap2 was released in the year of 2015.

Bootstrap4 was released in the year of 2019 by Mark Otto.

Bootstrap5 which is a extension of Bootstrap4 was officially released on May 5, 2021. 


Advantages of Bootstrap5
========================
1) It is simple to learn and use.

2) It is used to develop responsive designs.

3) It saves lot of development time.

4) It is a consistent framework.

5) Good documentation and community support.

6) It supports greate grid system.

7) It gives javascript plugins along with JQuery.
