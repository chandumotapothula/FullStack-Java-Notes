Bootstrap tables 
================
A table is a collection of rows and columns.

To represent bootstrap table we need to use .table class.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		
		<!-- add bootstrap plugins -->
		<meta charset="utf-8">
		<meta name="viewport" content="width=device-width, initial-scale=1">
		<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/css/bootstrap.min.css" rel="stylesheet">
		<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/js/bootstrap.bundle.min.js" ></script>
	</head>
	<body>
			<table class="table">
				<tr>
					<th>No</th>
					<th>Name</th>
					<th>Address</th>
				</tr>
				<tr>
					<td>101</td>
					<td>Alan</td>
					<td>USA</td>
				</tr>
				<tr>
					<td>102</td>
					<td>Jose</td>
					<td>UAE</td>
				</tr>
				<tr>
					<td>103</td>
					<td>Nelson</td>
					<td>UK</td>
				</tr>
				<tr>
					<td>104</td>
					<td>Lisa</td>
					<td>Candana</td>
				</tr>
			</table>
	</body>	
</html>

Bootstrap Alerts 
================
To create a bootstrap alert we need to use .alert class along with contextual classes.

We have following lists of contextual classes.

ex:
	alert-danger 
	alert-success
	alert-info 
	alert-warning 
	and etc

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		
		<!-- add bootstrap plugins -->
		<meta charset="utf-8">
		<meta name="viewport" content="width=device-width, initial-scale=1">
		<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/css/bootstrap.min.css" rel="stylesheet">
		<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/js/bootstrap.bundle.min.js" ></script>
	</head>
	<body>
			<div class="container mt-5">
				<div class="alert alert-info">
					<strong>Info!</strong> Please come regular to classes
				</div>
				<div class="alert alert-warning">
					<strong>Warning!</strong> No Practive No Job
				</div>
				<div class="alert alert-danger">
					<strong>Danger!</strong> No Job No Marriage
				</div>
				<div class="alert alert-success">
					<strong>Success!</strong> Finally Got the Job
				</div>
			</div>
	</body>	
</html>

Bootstrap list 
==============
A .list-group class is used to declare a bootstrap list.

A .list-group-item class is used to declare a bootstrap list of item.

We have following classes to add the color to bootstrap list of item.
ex:
	.list-group-item-primary 
	.list-group-item-secondary
	.list-group-item-info
	.list-group-item-warning
	.list-group-item-danger
	.list-group-item-success 

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		
		<!-- add bootstrap plugins -->
		<meta charset="utf-8">
		<meta name="viewport" content="width=device-width, initial-scale=1">
		<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/css/bootstrap.min.css" rel="stylesheet">
		<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/js/bootstrap.bundle.min.js" ></script>
	</head>
	<body>
			<div class="container mt-5">
				<ul class="list-group w-50 text-center">
					<li class="list-group-item"> HTML </li>
					<li class="list-group-item"> CSS </li>
					<li class="list-group-item"> JavaScript </li>
					<li class="list-group-item"> Bootstrap </li>
				</ul>
			</div>
	</body>	
</html>

Bootstrap Badge
=================
A .badge class is used to declare bootstrap badge.

It is used to highlights the notifications.

It is used to add additional information to any content.

ex:
---

<html>
	<head>
		<title>MyPage!</title>
		
		<!-- add bootstrap plugins -->

	</head>
	<body>
		<div class="container mt-5">
			<ul class="list-group w-50">
			<li class="list-group-item"> 
				HTML <span class="badge bg-primary">Basic</span>

			<li class="list-group-item">

Bootstrap Progress Bar 
======================
A progress bar is a visual component that indicates the progress of a task or process.
ex:
	downloading a file 
	installing a software 
	transfering a data 
	and etc. 

We use the .progress as a wrapper to indicate the max value of the progress bar.

We use the inner .progress-bar to indicate the progress so far.

The .progress-bar requires an inline style, utility class, or custom CSS to set their width.

ex:
---
<!DOCTYPE html>

	</head>
	<body>
		<div class="container mt-5">

			<div class="progress">
				<div class="progress-bar" style="width:50%"> 50% </div>
			</div>

		</div>
	</body>	
</html>

:root
============
:root is a CSS pseudo-class selector that targets the highest-level element in the document — usually the <html> element.

It’s often used to define CSS variables that should be accessible globally.


ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		
		<!-- add bootstrap plugins -->
		<meta charset="utf-8">
		<meta name="viewport" content="width=device-width, initial-scale=1">
		<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/css/bootstrap.min.css" rel="stylesheet">
		<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/js/bootstrap.bundle.min.js" ></script>

		<!-- Bootstrap Font Icon CSS -->
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.5.0/font/bootstrap-icons.css">

    <style>
    	:root
    	{
    		--primary-color : #FF0000;
    		--secondary-color : #FFFF00;
    		--lime-color : #00FF00;
    	}
    	
    	h1
    	{
    		color: var(--primary-color);
    	}
    	p
    	{
    		color: var(--lime-color);
    		background-color: var(--secondary-color) ;
    	}

    </style>


	</head>
	<body>
		<h1> This is Heading Tag </h1>
		<p> This is Paragraph Tag </p>
	</body>	
</html>
