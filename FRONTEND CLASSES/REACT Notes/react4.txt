state 
======
It is similar to props but it is private and fully controlled by a component.

A state can be updatable or modified where as props is read only.

There are two ways to initialize the state in react.

1) Inside a class 

2) Inside a constructor 

1) Inside a class
=================
We can intialize the state inside a class as follow.
ex:

import {Component} from 'react';
class App extends Component
{
	state = 
	{
		-
		-
	}
	render()
	{
		return ()
	}
}
export default App;

    }

Q) What is the difference between props and state ?

props					state 
------					------
It is immutable.			It is mutable.

It is read only.			It is updatable.

Stateless component can have props.	Statefull component can have state.

We can share props from one component	It is controlled by a component itself.
to another component as an argument.

A child component can access props.	It is private.  
					
Event Handling in React 
========================
Event handling for react elements is similar to event handling of DOM elements.

Action to which javscript response is called event.

ex:
	JavaScript 
	----------
		<button onclick="f1()"> clickme </button>
	ReactJS 
	--------
	function component 
		ex:
			<button onClick={handleClick}> clickme </button>
	class component 
		ex:
			<button onClick={this.handleClick}> clickme </button>

-------
import React, { Component } from 'react'

export default class App extends Component 
{
  state = 
  {
      rollno : 101,
      name : "Alan"
  }

  buttonStyle=
  {
    padding: "8px",
    width:"200px",
    backgroundColor:"red",
    border:"none",
    borderRadius:"15px"
  }

  handleClick = (e)=>{
    e.preventDefault();
    this.setState({rollno:601,name:"Nelson"});
  }

    render() 
    {
      return (
        <>
           <center>
                <h1>Roll No : {this.state.rollno} </h1>
                <h1>Name : {this.state.name} </h1>
                <button style={this.buttonStyle} onClick={this.handleClick}> clickme </button>
           </center>
        </>
    )
  }
}

Sealed class in Java (17 version)
==================================
sealed class Parent permits Child1,Child2 
{
	public void property()
	{
		System.out.println("property-method");
	}
}
non-sealed class Child1 extends Parent 
{
	public void profession()
	{
		System.out.println("Doctor");
	}
}
non-sealed class Child2 extends Parent 
{
	public void profession()
	{
		System.out.println("Jobless");
	}	
}
class Test 
{
	public static void main(String[] args)
	{
		Child1 c1 = new Child1();
		c1.property();
		c1.profession();
		
		
		Child2 c2 = new Child2();
		c2.property();
		c2.profession();
	
		
	}
}
