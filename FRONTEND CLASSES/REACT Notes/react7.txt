React Router 
============
React router is a standard library system built on top of react to perform routing in react application using react-router package.

Routing is a process in which a user is redirected to other pages based on the actions and requests.

React router is used to develop Single Page Applications (SPA).

React router is used to declare multiple routes in routing file.

If user entered any url in browser address if given path matches to any route then that request will be forwarded to perticular route.

There three packages to use react router.

1) react-router 
---------------
	It provides core components and functions to perform react routing. 

2) react-router-native 
----------------------
	It is used for mobile development.

3) react-router-dom 
------------------
	It is used for web development.

We have two component of react router.

1) <BrowserRouter>
---------------
	It is used to handle dynamic url's.

2) <HashRouter>
----------------
	It is used to handle static url's.

              <Route exact path='/' element={<Home/>}></Route>
              <Route path='/about' element={<About/>}></Route>
              <Route path='/service' element={<Service/>}></Route>
              <Route path='*' element={<Error/>}></Route>
          </Routes>
        </BrowserRouter>
    </div>
  )
}

export default App




React Forms 
===========
Forms are the integral part of every modern application.

Form allows users to interact with application and gather the user inputs.

Forms can be used for different purpose based on the nature of a business like registration form , order form , admission form and etc.

React offers statefull, reactive approach to develop the react forms.

In react forms are created by using controlled components.

A component rather then DOM handles the form.

App.jsx
---------
import { useState } from "react";
function App()
{
  const [registration,setRegistration]=useState({
    username: "",
    password:"",
    date:"",
    category:""
  })

  const [errors,setErrors]= useState({});


  const handleClick=(e)=>{
    e.preventDefault();
    const {name,value} = e.target;
    setRegistration({...registration,[name]:value});
    setErrors({...errors,[name]:""})
  }

  const handleSubmit = (e)=>{
    e.preventDefault();
    const formErrors = validation();
    if(Object.keys(formErrors).length>0)
    {
      setErrors(formErrors);
      return;
    }
    setRegistration({username:"",password:"",date:"",category:""});
    setErrors({});
  }

  const validation=()=>{
    const formErrors = {};
    if(!registration.username)
    {
        formErrors.username = "username is mandatory";
    }
    if(!registration.password)
    {
      formErrors.password = "Password is mandatory";
    }
    if(!registration.date)
    {
      formErrors.date = "Date is mandatory";
    }
    if(!registration.category)
    {
      formErrors.category="Category is mandatory";
    }

    return formErrors;
  }

  return (
    <>
      <div className="card w-50 p-3 bg-info mx-auto">
        <h4 className="text-center"> User Details </h4>
        <form onSubmit={handleSubmit}>
            <label className="my-2">UserName:</label>
            <input type="text" name="username" id="username" className="form-control"
            value={registration.username}
            onChange={handleClick}/>
            {errors.username && <small className="text-danger">{errors.username}</small>}
            <br/>

            <label className="my-2">Password:</label>
            <input type="password" name="password" id="password" className="form-control"
            value={registration.password}
            onChange={handleClick}/>
            {errors.password && <small className="text-danger">{errors.password}</small>}
            <br/>

            <label className="my-2">DOJ:</label>
            <input type="date" name="date" id="date" className="form-control"
            value={registration.date}
            onChange={handleClick}/>
            {errors.date && <small className="text-danger">{errors.date}</small>}
            <br/>

            <label className="my-2">Category:</label>
            <select name="category" id="category" className="form-select"
            value={registration.category}
            onChange={handleClick}>

                <option value="">none</option>
                <option value="entertainment">Entertainment</option>
                <option value="action"> Action </option>
                <option value="drama"> Drama </option>
            </select>
            {errors.category && <small className="text-danger">{errors.category}</small>}
            <br/>
          <button className="btn btn-warning w-100 mt-4"> submit </button>
        </form>
      </div>
    </>
  )
}
export default App;
